---

## About

OOTD Mate is a digital wardrobe and outfit planning app designed to help users decide what to wear — without the stress. The app combines weather-aware layering suggestions, a visual outfit mixer, trend-matching from inspiration photos, and a personal outfit collection manager.

## Supported Tasks

| Task | Description |
|------|-------------|
| **Task 1 — Weather Layers** | Adjust your outfit based on today's weather conditions |
| **Task 2 — Outfit Mixer** | Swipe through wardrobe items to build a complete look |
| **Task 3 — Inspire Me** | Find an outfit style you love and see what you already own |
| **Task 4 — Collections** | Save and organize your favorite outfit combinations |

---

## Prototype Notes

- No backend — all data is canned/simulated
- Prototype supports a fixed demonstration path for each task
- Closet items can be browsed using the **‹ ›** arrow buttons
- Mixer rows respond to **left/right swipe** (mouse drag or touch)
